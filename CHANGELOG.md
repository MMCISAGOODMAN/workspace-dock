# Changelog

## [0.2.0] - 2026-06-28

### 新增

- **收藏应用**：本地应用与网页 URL 收藏，拖拽排序，Dock 栏火箭一键全部启动
- **剪贴板**：独立标签页展示复制内容，点击复制；与 SSH / 书签完全分离
- **区域截图**：`Cmd/Ctrl + Shift + S` 框选屏幕，自动复制到剪贴板
- **全局搜索增强**：支持搜索应用、快照；`>` 前缀快捷操作
- **临时收藏转书签**：一键归档到三维书签树
- **主机标签**：编辑主机时可添加搜索别名
- **快照导入**：与导出对称，支持 JSON 合并导入
- **面板 UI**：图标标签栏、紧凑浮动 Dock、面板展开动画

### 改进

- 临时 SSH 与剪贴板功能解耦，避免误触连接
- 增大 Dock 窗口高度与面板宽度，5 个标签完整显示
- 修复浅色主题、应用图标 CSP、macOS `.app` 启动等问题

### 下载

GitHub Release 提供以下安装包：

| 平台 | 格式 |
|------|------|
| macOS | `.dmg`、`.zip`（Intel + Apple Silicon） |
| Windows | NSIS 安装包、便携版 `.exe` |
| Linux | `.AppImage`、`.deb` |
